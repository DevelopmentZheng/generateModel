# flow based

给一张图片，训练一个encoder生成一个normal distribution ，并且要求encoder是可以翻转的，直接用这个翻转后的encoder去生成图片。

所以通过限制网络模型的结构 ，可以能马上知道反转结构得到模型。并且encode
